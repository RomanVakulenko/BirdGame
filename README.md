## BirdGame

**Features** 

- moving up-down
- collecting cherries
- taking points
- can be crached by monster

**Stack** 
- SpiteKit, GameplayKit
- Work with Assets, Sounds, Effects
- SKScene, SKPhysicsContactDelegate
